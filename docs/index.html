How it works

Your Mac creates a virtual 4:3 display, captures it with ScreenCaptureKit, converts to greyscale on CPU (vImage SIMD), delta-compresses with LZ4, and streams over USB to a native Android renderer. Every pixel is identical on both screens.

Virtual display

Creates a 4:3 display automatically using macOS private APIs. No BetterDisplay license needed. Your Mac thinks it's a real monitor.
